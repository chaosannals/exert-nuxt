# [exert-nuxt](https://github.com/chaosannals/exert-nuxt)

## 构建和安装

```bash
# 安装依赖
$ npm install

# 服务器会热加载 localhost:3000
$ npm run dev

# 构建发布版和启动服务
$ npm run build
$ npm run start

# 生成静态站点
$ npm run generate
```

更多详情, 请到 [Nuxt.js docs](https://nuxtjs.org).

## Visual Studio Code
